Instalar laravel

Configurar conexion a la BD
	DB_CONNECTION=sqlite
	#DB_HOST=127.0.0.1
	#DB_PORT=3306
	#DB_DATABASE=laravel
	#DB_USERNAME=root
	#DB_PASSWORD=

Crear archivo de sqlite
	database/database.sqlite

Ejecutar migraciones

Instalar laravel ui 
	composer require laravel/ui

Comando para generar estructuras (Vistas y funcionalidad de inicio de sesion)
	php artisan ui bootstrap --auth
	(resources/views/auth)

*Para modificar logica 
	app/Http/Controllers/Auth

Ejecutar npm install para instalar las dependencias

Ejecutar los script
	npm run dev

Configurar pusher

Instalar un libreria de pusher
	composer require pusher/pusher-php-server

Habilitar el provider
	config/app.php -> parte de provider
	App\Providers\BroadcastServiceProvider::class,

Instalar laravel echo y pusher-js
	npm install --save-dev laravel-echo pusher-js

Descomentar archivo bootstrap 
	npm run dev
